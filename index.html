Welcome to Arrows: The Game

This is a simple, yet challenging, game designed to test your reflexes.

  • Arrows will flash on the screen randomly
  • If it's a blue arrow, use the arrow keys to indicate the direction of the arrow
  • If it's a red arrow, use the arrow keys to indicate the opposite of the direction the arrow
  • Try to achieve the highest score possible in the given amount of time!
